# Sahil Kamble — Portfolio

This is a minimal portfolio website for **Sahil Gautam Kamble**, generated from the resume copy provided by the user.

## What’s included
- `index.html` — single-page portfolio
- `styles.css` — lightweight responsive styling

## How to preview locally
1. Clone the repo
2. Open `index.html` in your browser (or run a static server)

Example:
```bash
git clone https://github.com/ssk-animator/Website-.git
cd Website-
# double-click index.html or run:
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deploy to GitHub Pages
1. Push to a branch named `main` or `gh-pages`
2. In repo Settings → Pages, point to the branch root (`/`) and save
